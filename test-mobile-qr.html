📱 How to Test QR Code:

  1. Make sure your phone is on the same WiFi network as your computer
  2. Click "Test Connection" above
  3. If the test passes, try scanning the QR code again
  4. If the test fails, check your network settings

🔧 Troubleshooting:

  • Connection failed: Make sure both devices are on the same WiFi
  • QR code not loading: Try using the manual URL instead
  • Still not working: Contact your system administrator
